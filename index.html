What does "Powered by BNRP" mean?

BNRP stands for Bitcoin Name Resolution Protocol. BTCNative uses BNRP open resolution to help verify and display BTC-native name records, including owner data, address records, avatar or inscription links, primary identity records, and marketplace listing status.

BNRP resolves
  • name → owner address
  • name → primary Bitcoin address
  • name → avatar / inscription
  • name → text records
  • name → marketplace listing status
  • address → primary name (reverse lookup)
  • namespace → supported resolver / indexer

BNRP does not own the names. It is the resolver layer that helps BTCNative read and organize name data in a consistent way.
